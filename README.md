Print the largest files.

Usage of largest:
```
  -n, --lines int
  
    	Lines of output (default: 1)
     
  -d, --dir bool
  
    	Include directories (excluded by default)

  -r bool
      Reverse (print the smallest files)
```

Example:
```
./largest -n 3 -d ~/

Videos 66 GB

.local 32 GB

Games 29 GB
```
